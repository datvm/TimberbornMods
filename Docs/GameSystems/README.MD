# Game Systems

This folder documents Timberborn game systems and their observed behavior from the decompiled game sources in `ModdingAssetUpdater/out`.

Use these notes as quick references when working on mods. Each page should explain how a system behaves in practice, which core classes are involved, and what modding hooks or extension points are worth remembering.

## Pages

- [Character Behavior](CharacterBehavior.MD): how characters choose, prioritize, and execute behaviors.
