TImprove 4 Achievements adds optional hints to several Timberborn achievements:

- Certain hints to help you track your progress towards achievements.
- Reveal secret achievements if you choose to.
- Filter achievements by state (locked or unlocked) and keyword.
- Clear unlocked achievements: if you are up to the challenge again. This only clear the internal state so you can earn in-game again, if you play with Steam, it won't reset the Steam achievement progress.

[h1]Supported achievement hints[/h1]
[list]
[*]Show planted tree count
[*]Show remaining count of building types to build and List missing building types
[*]Show current 24h work day streak
[*]Show current power generated with only specified source
[*]Show current built hedges count
[*]Show current exploded dynamite count today
[*]Show plugged badwater sources count and total
[*]Show current produced planks count today
[*]Show current battery charge stored
[*]Show current longest zipline network length
[*]Show current built Station and Tubeway segments count
[/list]
