TImprove enhances your Timberborn experience by providing various quality-of-life improvements. These tweaks aim to provide a more seamless and customizable gameplay experience.

[b]By default, only "Show game time" and "Enable 30x speed" are enabled, you need to check the options in the Mod Settings to enable them[/b], including those with shortcut (for example, the Quick Quit [Ctrl + Shift + Q] shortcut won't work until you check the option).

[h1]Camera & View[/h1]

[b]Enable Free Camera[/b]: Removes camera restrictions, allowing unlimited zoom and rotation.
+ [b]Lock Free Camera Angle[/b]: When free camera is enabled, keeps the vertical angle locked to prevent unwanted tilting.
[img]https://files.catbox.moe/xiuk2e.jpg[/img]

[b]Disable Fog[/b]: Remove the fog in the game.
[img]https://files.catbox.moe/g5lfok.jpg[/img]

[b]Static Sky and Sun Light[/b]
Visual/UI only. Locks the sun and sky lighting to a fixed state.
[img]https://files.catbox.moe/kvsbqc.jpg[/img]

[h1]Extra info[/h1]

[b]Show Coordinates[/b]: Displays block coordinates in the top-right corner of the screen. Z represents height.
+ [b]Only Show the Height[/b]: Shows only the height value instead of full coordinates.
[img]https://files.catbox.moe/cmlc0p.jpg[/img]

[b]Show Game Time[/b]: Displays the current in-game time in the top-right corner.
[b]Add Real-Time Clock[/b]: Adds a real-world clock to the top-right corner.
+ [b]Use 24-hour Format[/b]: Displays time in 24-hour format (e.g. 17:30 instead of 05:30).
[img]https://files.catbox.moe/b7esk2.jpg[/img]

[h1]Automation & Priorities[/h1]

[b]Prioritize Rubbles[/b]: Automatically sets rubble gathering priority to High, ensuring builders collect materials first.
[img]https://files.catbox.moe/ah60t6.jpg[/img]

[b]Auto Clear Dead Stumps[/b]: Automatically clears dead stumps or bushes that could normally be removed by building over them.
[img]https://files.catbox.moe/pg9rph.jpg[/img]

[b]On New Day[/b]: Automatically performs an action when a new day starts. You can choose one of the following behaviors:
- None: do nothing
- Pause: pause the game
- Speed 1: set the game speed to 1x

This can be configured to trigger on:
- Every day
- Weather warning days
- Hazardous weather begin days
- New cycle days

[b]On Shift End[/b]: Automatically changes the game speed when the work shift ends each day. You can choose:
- Do nothing
- Set the game to a specific speed

[h1]Game Speed[/h1]

[b]Enable 0.25x Speed [Shift+1][/b]: Unlocks the 0.25x slow speed option.
[b]Enable 30x Speed [4][/b]: Unlocks the 30x speed option.
[b]Enable 99x Speed [5] (!)[/b]: Unlocks the 99x speed option. [b]⚠️ Warning:[/b] This speed is unstable and may cause issues such as beavers not eating or drinking properly.
[img]https://files.catbox.moe/vdkr9z.jpg[/img]

[h1]Building[/h1]

[b]Bigger Build Drag Area (!)[/b]: Increases the maximum drag size for buildings like Dams and Roads. Example: setting this to 5 increases Dam drag size from 10x10 to 15x15. [b]⚠️ Warning:[/b] Significantly increases load time and memory usage. [b]Requires reloading a save to take effect.[/b]
[img]https://files.catbox.moe/vout9x.jpg[/img]

[h1]Misc[/h1]

[b]Quick / Force Quit [Ctrl + Shift + Q] (!)[/b] Immediately terminates the game process. [b]⚠️ Warning:[/b] No saving or cleanup is performed. Use with caution.
