This is a support mod for other mods that prevent certain crashes from happening due to the game's constraints. It contains the following features:

- [b]MaterialPlzNoCrashes[/b]: Duplicate materials should not crash but print a warning in the log instead.
- [b]XtraNeedPlzNoCrashes[/b]: Needs and NeedCollections that are not used will no longer crash.
- [b]MultiYieldersPlzNoCrashes[/b]: Duplicate gatherable should not crash the game.
- [b]Duplicate Blueprint PlzNoCrashes[/b]: Duplicate blueprints from multiple template collections should not crash the game. Implemented by [url=https://github.com/datvm/TimberbornMods/pull/14]normanr[/url].
- [b]XtraComponentSpecPlzNoCrashes[/b]: Extra undeclared ComponentSpecs in blueprints should not crash the game. This essentially make optional ComponentSpec possible. A warning is still printed.
